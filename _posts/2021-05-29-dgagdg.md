# 당분간의 공부 계획

싸피에서의 1학기를 마무리하고 일 주일간의 방학을 맞이하였다. 집안 일도 좀 있고 해서 오롯이 부족한 공부를 하는데 시간을 쓸 수 없지만 최대한 시간을 활용해볼까 한다.

1. REST API 공부(feat. Django REST Framework) 

   이번 프로젝트를 진행하면서 REST API에 대한 그 자체와 클라이언트와 대화하는 것에 대한 이해가 부족하였고, 이러한 이해 부족은 Django REST Framework를 사용하는 것에 어려움을 가져왔다. 내가 실무에서도 Django를 쓸지는 알 수 없으나, 이 자체는 도구일 뿐 Django REST Framework라는 도구를 활용하여 REST API에 대한 이해와 프로그래밍 실력을 키우는데 1순위로 시간을 쓸 생각이다.

2. C#을 활용한 채팅 서버 구현

   한동안 C#의 기본 문법을 들여다보고 채팅 서버 구현에 필요한 것들을 틈틈히 공부중이었는데 이번 프로젝트를 진행하면서 잠시 미뤄두고 있었다. 이제 시간적 여유가 약간 생겼으니 다시 공부를 시작해볼까 한다. 이번 프로젝트에서 웹소켓 프로그래밍을 처음 해봤는데, 사실 동기와 비동기에 대한 이해도 부족했고, 이를 구현하는 부분에서도 부족함을 느꼈다. 아직 갈 길이 멀다는 것을 느꼈기에 좀더 끈덕지게 이 분야를 정진해볼 생각이다. 특히 멀티스레딩 분야는 참 흥미로운 것 같다.

3. 알고리즘 문제 풀이

   프로젝트를 준비하고, 채팅 서버 관련된 것들을 준비하면서 한동안 PS를 많이 못 했다. 다시 하루에 최소 한 문제 정도를 풀 생각이다. 당분간 좀 풀어보고 싶은 것은 BFS, 그래프, 트리와 구현 파트이다.
